# Config

Personal repo where I save my config files for various things:
### **tmux**:
  - install `https://github.com/tmux-plugins/tpm`
  - mkdir -p ~/.config/tmux/plugins/catppuccin
  - install catppuccin theme for tmux`git clone -b v2.1.3 https://github.com/catppuccin/tmux.git ~/.config/tmux/plugins/catppuccin/tmux`
  - Copy the content of this repo into ~/.tmux.conf
  - install plugins `Press Ctrl-s + I` in a tmux session
  - NOTE: It works on linux(Pop os) & macos
### **zshrc**
  - install `https://ohmyz.sh/#install`
